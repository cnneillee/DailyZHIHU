# DailyZHIHU

一款基于知乎日报内容的第三方应用

## API
基于第三方 API [izzyleung/ZhihuDailyPurify][0]

## Declaration
- 1. 关于内容
内容来自知乎日报，APP 基于 [izzyleung/ZhihuDailyPurify][0] 提供的第三方 API。

- 2. 关于版权
项目为练习之用，如有侵权，本人将于第一时间删除。

- 3. 关于引用
您可以随意使用本项目中的源码，烦请注明出处。

如有问题，欢迎交流！
